css: styling the content

this Makes website alive with good look

we can style html in three ways:

  1. inline
    • this is applied to each element in the html page itself
  2. internal
    • this is applied to html page in the head section
  3. external
    • this is applied via external css file linked in the head section

Highest priority

Inline Style

Not suitable for all the page without using external or internal styles

Internal Style

</head>

External Style

<head>

</head>
